# Infrastructure

`infra/` contains deployment and operational scaffolding.

`docker/` is for local/container runtime.

`k8s/` is for future Kubernetes manifests.

`observability/` is for OpenTelemetry, Prometheus, Grafana, Loki/ELK/Jaeger style configuration over time.

`local_dev/` is for local-first development helpers.

No production deployment logic is added in this patch.
