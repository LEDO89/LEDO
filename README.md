# ledo_ontology_core

Ontology-centric cyber-physical platform core.

This repository is designed as both:

1. a specification repository,
2. a Codex-ready implementation workspace,
3. a future Python package,
4. an ontology engineering skill portfolio.

Core rule:

```text
Codex builds the structure.
The human domain expert provides the meaning.
Validators protect the meaning.
Tests prove the behavior.
```
